# Project Overview

Module `gophers.dev/cmds/envy` provides a command-line utility for managing
secretive environment variables when running commands.

# Getting Started

#### Build from source

The `envy` command can be compiled by running
```bash
$ go get gophers.dev/cmds/envy
```
